# «Разработка модели для поддержки принятия решений при диагностике кожных заболеваний на основе изображений и описаний симптомов

## Возможности

- **Анализ изображения** — классификация с помощью EfficientNet-B0, обученного в финальной версии на SD-198
- **Анализ симптомов** — Random Forest, предсказывающий группу заболевания по клиническим признакам
- **Гибридный режим** — объединение оценок изображения и симптомов: вероятность диагноза усиливается, если симптоматическая группа совпадает с группой диагноза
- **Проверка качества изображения** — фильтрация по размытости, яркости, наличию кожи и признакам скриншота

## Структура проекта

```
VKR/
├── api/                    # FastAPI-приложение + HTML-шаблоны
│   └── app.py
├── src/skin_dss/           # Основная библиотека
│   ├── data/               # Dataset-классы и векторизатор симптомов
│   ├── inference/          # Предикторы, маршрутизатор, гибридное слияние
│   └── utils/              # Названия диагнозов, рекомендации
├── scripts/                # Скрипты обучения, оценки и визуализации
├── models/                 # Сохранённые веса моделей (не в репозитории)
├── experiments/            # Результаты экспериментов (JSON, CSV, PNG)
├── data/                   # Датасеты (HAM10000, Skin-Extra, SD-198) (не в репозитории)
└── symptoms_dataset_sd198.csv
```
## Используемые датасеты

| Датасет | Классов | Назначение |
|---------|---------|------------|
| HAM10000 | 7 | Базовая модель для дерматоскопических изображений |
| Skin-Extra |9 | Для расширения набора заболевани |
| SD-198 | 198 | Большой охват, фмнальный датасет |

## Запуск 

```bash
uvicorn api.app:app --reload
```

Система будет доступна по адресу `http://localhost:8000`.

## Режимы работы

| Режим | Условие | Результат |
|-------|---------|-----------|
| Только изображение | загружено фото | Top-3 диагноза от SD-198 |
| Только симптомы | заполнены симптомы | Top-3 групп заболеваний |
| Гибридный | фото + симптомы | Top-3 диагноза с усилением по клинической группе |
